HOW TO DELETE ROW?
syntax 
    delete from <tablename> [<condition>]
1) delete all row from table 
    delete from products 

2) delete specific row from table 
   Question : delete those row whose product id is 1 
   SQL : delete from products where id = 1 

   Question : delete those row whose product id in range of 10 to 20
   SQL : delete from products where id >= 10 and id <= 20
 
   Question : delete those row from product table of Navneet brand 
   SQL : delete from products where brand = 'Navneet'
   
   Question : delete those row from product table of whose product_code is PRD0001 or PRD0002
   SQL : delete from products where product_code='PRD0001' or product_code='PRD0002'

How to remove all row from table and reset primary key?
truncate table <tablename>
truncate table products

How to update existing row into table?
syntax 
update <tablename> set <fieldname=value> [condition]

Examples 
    Question : - update all product increase price by 1.99 rupees 
    update products set price=price+1.99

    Question : update product set its price to 99 55 whose product id is 1 
    update products set price = 99 where id = 1


    Question : update product set its name to "Bharat Pencil" brand "india" price 50 rupees of those product whose product code is PNCL1001
    update products set name="Bharat Pencil",brand="India",price=50 where product_code='PNCL1001'
How to rename table?
    rename table <tablename> to <newtablename>
    rename table products to product

How to delete table?
    drop table <tablename>
    drop table product

How to Fetch Data from table?
------------------------------------------------------------------------------
syntax 
SELECT * | column_list FROM table_name [WHERE condition] [GROUP BY column_name] [HAVING condition] [ORDER BY column_name] [ASC|DESC LIMIT number];

max()
max() function returns the maximum value of given field

count()
